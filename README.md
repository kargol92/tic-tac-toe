Tic-Tac-Toe
===========
Tic-Tac-Toe is a simple game written in HTML5, CSS and JavaScript.

Game - [karol-kiersnowski.github.io/tic-tac-toe](https://karol-kiersnowski.github.io/tic-tac-toe)

Technologies
------------
* HTML 5
* CSS
* JavaScript


Features
--------
CSS webfonts
* Finger Paint font
* Font Awesome icons

Language
--------
* User interface: English / Polish
* Source code: English

Requirements
------------
* Web browser
