tic-tac-toe
===========
Tic-tac-toe is a simple game written in HTML5, CSS and JavaScript.

Project site - [cmd.sh](http://karol-kiersnowski.prv.pl/#tic-tac-toe)

Technologies
------------
* HTML 5
* CSS
* JavaScript


Features
--------
CSS webfonts
* Finger Paint font
* Font Awesome icons

Language
--------
* User interface: English / Polish
* Source code: English

Requirements
------------
* Web browser
